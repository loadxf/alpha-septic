Icons Required for Alpha Septic Website

The following icon files need to be created from the logo.webp file found in the public directory:

  • favicon.ico - Multi-size icon file (16x16, 32x32, 48x48)
  • icon-192.png - 192x192 PNG icon
  • icon-512.png - 512x512 PNG icon
  • icon-maskable.png - 512x512 PNG icon with padding for maskable icon support

Instructions for Generation

Use an online converter like CloudConvert or Convertio to convert the logo.webp file to the required formats.

For the maskable icon, ensure there is sufficient padding (about 15% from each edge) to allow for safe display on various platforms.

NOTE: After generating these icons, place them in the public directory for favicon.ico and the public/icons directory for the PNG files.

Icon Usage

These icons are referenced in the following files:

    - app/manifest.ts: All icons are referenced here
    - app/layout.tsx: favicon.ico is referenced

Once all icons are generated and placed in the correct locations, this placeholder file can be removed.
